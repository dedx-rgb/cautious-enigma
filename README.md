# ARMS FIT // PREMIUM TACTICAL & COMBAT APPAREL

ARMS FIT is a premium, military-inspired e-commerce platform engineered with a futuristic "Anti-Gravity" tactical HUD (Heads-Up Display) user interface. Built entirely with Vanilla HTML5, CSS3, and JavaScript, it offers high product visibility, instant performance-driven interactions, and deep visual customizability.

---

## 🛰️ SYSTEM FEATURES

### 1. Camouflage Theme Selector
A dynamic interface selector in the navigation actions menu overrides CSS custom properties across the site:
*   **SYS: MULTICAM BLACK (Default)**: Deep matte black layout with neon active cyan alerts.
*   **SYS: WOODLAND GREEN**: High-contrast forest theme with active lime-green borders.
*   **SYS: DESERT COYOTE**: Desert clay accents styled with amber-glow indicators.
*   **SYS: LIGHT MODE**: A complete light-themed template adapting headers, overlays, footer panels, and slide-drawers into a highly readable, high-contrast visual interface.

### 2. Adaptive Illumination Tracking Spotlight
A cursor-tracking spotlight tracks client navigation on the page:
*   **Dark Themes**: Emits a faint glowing neon-green laser illumination spotlight tracking user mouse coordinates.
*   **Light Mode**: Automatically converts into a subtle dark shadow tracking spotlight, preserving visual depth.
*   *Exclusivity*: When in Light Mode, moving the mouse over headers, footers, modal details, and side-drawers (which are styled to maintain high-contrast legibility) dynamically restores the green laser spotlight.

### 3. Mission Manifest (Shopping Cart)
*   Equipped with a slide-out panel drawer tracking active product quantities, subtotal calculations, and item removals.
*   Features local storage persistence, ensuring manifest parameters remain saved between operational sessions.

### 4. Watchlist Monitor (Wishlist)
*   A dedicated tracker allowing operators to bookmark tactical components for replenishment.
*   Enables transferring bookmarked products directly into the active checkout manifest with a single click.

### 5. System Menu Drawer (Left Panel)
Accessible via the header toggle `[MENU //]`, this HUD panel provides:
*   **Operational Profile (About Us)**: A brand profile outlining ARMS FIT's design philosophy, material specifications (1000D Cordura, eVent membranes, titanium buckles), and collaboration with veteran detail details.
*   **Inventory Index**: A compact inventory index allowing users to sort catalog products in real-time by **Top Operator Rated**, **Price: High to Low**, **Price: Low to High**, or **Alphabetical: A-Z**.
*   **Direct Linkage**: Selecting a product from this menu closes the drawer and displays the product's detail modal instantly.

### 6. Interactive Quick-View Modal
*   Displays comprehensive product details, bicep storage grids, size configurations, and technical lists.
*   Allows operators to swap between front lifestyle photography and detailed wireframe HUD vector blueprint representations of the equipment.
